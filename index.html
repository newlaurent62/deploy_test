Bienvenue sur Static Site Deployer

Déployez facilement vos sites statiques sur diverses plateformes.

Qu'est-ce que Static Site Deployer ?

Static Site Deployer est un outil générique qui vous permet de déployer des sites web statiques sur plusieurs plateformes de manière simple et rapide.

Fonctionnalités :

  • GitHub Pages : Déployez votre site sur GitHub Pages avec une simple commande.
  • GitLab Pages : Déployez votre site sur GitLab Pages.
  • SFTP : Déployez votre site sur un serveur SFTP.
  • S3 : Déployez votre site sur un bucket AWS S3.
  • Docker : Déployez votre site dans un conteneur Docker.

Comment l'utiliser ?

1. Clonez le repository du projet ou téléchargez les fichiers nécessaires.

2. Assurez-vous d'avoir Python et les dépendances requises installées (voir requirements.txt).

3. Utilisez la ligne de commande pour exécuter le déploiement :

python deployer.py --platform  --user  --repo-url  --local-path  --domain

Exemples :

python deployer.py --platform github --user monUtilisateur --repo-url https://github.com/monUtilisateur/monRepo --local-path ./build --domain monsite.com
